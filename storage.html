ABC's of Storage

  • AWS provides a range of services for storing and accessing data for cost optimization and increased data availability.
  • There are multiple types of data storage options with additional backup and data migration facilities.

Data Storage Categorization

Block Storage File Storage Object Storage
Data is stored in ‘blocks’. Data is stored as ‘files’ in a series of directories. Each ‘object’ is stored across a flat address space.
Blocks are stored on a volume and attached to a single instance. Data is stored in a file system. Data is stored in objects, which are in turn stored in buckets.
Provides very low latency data access. Provides shared access for multiple users. Provides access to a large number of users simultaneously.
